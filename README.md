# TestSystemForJava

JAVA课程设计项目--JAVA在线考试系统。

## 运行环境

### 开发工具包

jdk1.8.0_40

### 开发环境

Notepad++




## 主要模块

### 登录模块
登录模块分成用户和管理员，分为两个数据库，互相不干扰。
对于新用户，还添加了注册模块。
管理员不能注册，管理员账号密码为系统初始设置。
考生用户登录将进入考试系统。
管理员登录将进入后台题库管理系统。

### 管理后台模块
管理员模式用正确的管理员密码可以登入后台模块。对题目进行基础的增、删、改、查。

### 考试模块
用户模式登录可以进入考试模块，题目从后台题库随机选取生成。

### SQL预处理模块
主要用来完成SQL语言的预处理，包括加载数据库驱动，连接数据库，搜索，查询，删除的SQL语句的预处理。

## 使用方法

安装完jdk1.8.0_40 (X32)后, 配置JAVA和Derby数据库的环境，具体可以百度，不再累述。
配置完环境后，在文件目录下命令行窗口输入：
```
javac *.java
java SystemLogin
```

如果只想看源代码，在```src``` 目录下。

如果遇到以下驱动加载问题：
```
java.lang.ClassNotFoundException: org.apache.derby.jdbc.EmbeddedDriver
```
请查看是否将```%JAVA_HOME%\db```添加到相关环境路径。
如果路径没有问题，请同时确保文件夹不在C盘下运行(无法获得权限)，同时修改文件夹属性，不要勾选只读。

用户登录请先注册，用注册的账号密码登录。

管理员登录，请输入：
```
账号：admin
密码：mima
```
